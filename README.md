# Automatic Zoom

نموذج مشروع متكامل لمادة IT505 تم بناؤه باستخدام .NET 10 وAngular وEntity Framework Core بنمط code-first. يوضح التطبيق واجهة مناسبة لمبادئ HCI بألوان واضحة وخطوط مريحة، مع بحث عن الموظفين بالاسم أو الكود أو تاريخ التعيين، ونماذج إدخال مخصصة لكل جدول، وحوار تأكيد للحذف، وتنبيهات للحالات المهمة.

## التقنيات المستخدمة
- الخلفية: .NET 10 ASP.NET Core Web API
- ORM: Entity Framework Core code-first
- قاعدة البيانات: SQLite
- الواجهة الأمامية: Angular 20

## أهم المزايا
- البحث عن الموظفين بالاسم أو الكود أو تاريخ التعيين
- عرض النتائج داخل جدول بيانات
- إضافة وتعديل الأقسام
- إضافة وتعديل الموظفين
- تأكيد الحذف قبل التنفيذ
- عرض تنبيهات للنجاح والتحذير والمعلومات والأخطاء
- إنشاء قاعدة البيانات تلقائيًا عند تشغيل الخلفية لأول مرة

## تشغيل المشروع
1. تشغيل الخلفية:
   ```bash
   dotnet run --project backend
   ```
2. تثبيت حزم الواجهة الأمامية إذا لزم الأمر:
   ```bash
   cd frontend
   npm install
   ```
3. تشغيل الواجهة الأمامية:
   ```bash
   npm start
   ```

## النشر المجاني المقترح
- أنسب طريقة مجانية وثابتة لهذا المشروع هي تشغيل الواجهة والخلفية على نفس الدومين في الإنتاج، مع توجيه `/api` إلى الخلفية.
- ابنِ الخلفية أولًا:
   ```bash
   dotnet publish backend/IT505.Api.csproj -c Release -o publish
   ```
- ابنِ الواجهة الأمامية:
   ```bash
   cd frontend
   npm install
   npm run build
   ```
- انشر ناتج Angular الثابت مع خدمة ASP.NET Core خلف Nginx أو أي reverse proxy مجاني على VM مجاني دائم، ثم وجه الجذر ` / ` للواجهة و`/api` للخلفية.
- إذا أردت استضافة الواجهة على دومين منفصل لاحقًا، غيّر قيمة `apiBaseUrl` في ملف الإنتاج داخل `frontend/src/environments/environment.production.ts`.

## الروابط الافتراضية

- التطبيق الأمامي: http://localhost:4200
