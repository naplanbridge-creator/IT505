# مخطط التقرير

## 1. تعريف المشكلة
- يقدم المشروع نموذجًا أوليًا لنظام إدارة بواجهة نظيفة وسهلة الاستخدام.
- يدعم البحث والإدخال وتأكيد الحذف والتنبيهات الخاصة بتغذية المستخدم.

## 2. الأهداف
- بناء واجهة واضحة ومناسبة لمبادئ HCI.
- استخدام .NET 10 وAngular وEF Core بنمط code-first.
- حفظ البيانات وإدارتها داخل طبقة قاعدة بيانات آمنة.
- عرض وظائف CRUD مع التصفية في البحث.

## 3. لغات البرمجة وبيئة التطوير
- C# مع ASP.NET Core Web API
- TypeScript مع Angular
- HTML وSCSS وتهيئة JSON
- Visual Studio Code

## 4. تصميم قاعدة البيانات
- جدول Department
- جدول Employee
- علاقة واحد إلى متعدد بين Department وEmployee
- راجع `database-erd.md` لمخطط ERD

## 5. لقطات الشاشة المطلوب تضمينها
- الشاشة الرئيسية
- نموذج إدخال القسم
- نموذج إدخال الموظف
- نموذج البحث مع الفلاتر
- جدول نتائج البحث
- نافذة تأكيد الحذف
- تنبيهات النجاح والخطأ

## 7. الخلاصة
- اشرح ما الذي يحققه النموذج الأولي.

- في نسخة الإنتاج تُخدم الواجهة والخلفية من نفس الدومين، ويُستخدم المسار النسبي `/api` للوصول إلى الخلفية.
