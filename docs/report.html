1. تعريف المشكلة

يقدم المشروع نظامًا بسيطًا لإدارة الأقسام والموظفين بواجهة عربية واضحة وسهلة الاستخدام.

2. الأهداف

  • بناء واجهة واضحة ومتجاوبة.
  • استخدام .NET 10 وAngular وEF Core بنمط code-first.
  • تنفيذ وظائف CRUD والبحث والتصفية.
  • عرض النموذج بشكل مناسب للتسليم النهائي.

3. تصميم قاعدة البيانات

  • جدول Department
  • جدول Employee
  • علاقة واحد إلى متعدد بين Department وEmployee
