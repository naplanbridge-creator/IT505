# الواجهة الأمامية

تم إنشاء هذا المشروع باستخدام Angular CLI.

## تشغيل بيئة التطوير

لتشغيل الخادم المحلي استخدم:

```bash
ng serve
```

بعد تشغيل الخادم افتح المتصفح على العنوان التالي:

```text
http://localhost:4200/
```

سيتم تحديث التطبيق تلقائيًا عند تعديل أي ملف من ملفات المصدر.

## إنشاء المكونات

لإنشاء مكوّن جديد استخدم:

```bash
ng generate component component-name
```

ولعرض جميع الأوامر المتاحة استخدم:

```bash
ng generate --help
```

## بناء المشروع

لتجميع المشروع استخدم:

```bash
ng build
```

سيتم حفظ ناتج البناء داخل مجلد `dist/`.

## الاختبارات

لتشغيل اختبارات الوحدة استخدم:

```bash
ng test
```

ولتشغيل اختبارات النهاية إلى النهاية استخدم:

```bash
ng e2e
```

## مراجع إضافية

لمعرفة المزيد عن أوامر Angular CLI يمكنك الرجوع إلى التوثيق الرسمي الخاص به.
